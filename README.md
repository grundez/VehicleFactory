## Описание

### Старт
**Клонируйте репозиторий**
```cmd
git clone https://github.com/grundez/VehicleFactory.git
cd VehicleFactory
```

---
### Сборка
**Для Windows:**
```cmd
chmod +x build.sh
./build.sh
```
**Для Linux/Mac**
```cmd
build.bat
```

---
### Работа с программой


Программа принимает на вход числа в диапазоне 0-3:  
0 - Мотоцикл  
1 - Самокат  
2 - Автомобиль  
3 - Автобус


**Windows**
```cmd
build\Release\TransportProject.exe <аргументы>
```

**Linux/Mac**
```cmd
./build/TransportProject <аргументы>
```
